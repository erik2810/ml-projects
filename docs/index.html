05 — Mesh VAE

Low-Poly Mesh Latent Space Interpolation

A graph VAE with inner-product decoder (VGAE) extended to jointly predict 3D positions. Enables smooth interpolation between meshes of different topology in a shared latent space.

Procedural Meshes

Six distinct low-poly shapes: Platonic solids, prisms, stellated forms, and a torus. Each mesh has unique topology and vertex geometry.
